# 🤖 بوت إدارة حضور الموظفين

بوت تليجرام متكامل لإدارة حضور الموظفين وتنظيم العمل.

## ✨ الميزات

### للموظفين
- ✅ تسجيل الحضور والانصراف
- 🚬 استراحات تدخين ذكية
- 🍽️ استراحات غداء
- 📅 طلبات إجازة
- 📊 تقارير حضور شخصية

### للمديرين
- 👥 إدارة الموظفين
- 📈 تقارير الحضور اليومية والأسبوعية
- 👨‍💼 إدارة المديرين
- ✅ الموافقة على طلبات الإجازة
- ⚠️ نظام إنذارات
